# Code of Conduct

This project follows a simple standard: be direct, respectful, and useful.

## Expected Behavior

- Report bugs with enough detail to reproduce them.
- Challenge ideas with technical reasoning, not personal attacks.
- Keep discussions focused on making repositories easier for AI coding agents to work in safely.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Sharing private code, secrets, or credentials in public issues
- Spam or unrelated promotion

Maintainers may edit, hide, or close interactions that make the project less useful for contributors.
